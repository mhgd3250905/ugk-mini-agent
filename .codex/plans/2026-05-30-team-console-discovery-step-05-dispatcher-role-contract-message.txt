- Dispatcher output JSON 必须是：
  {"itemId":"exact_item_id","workUnit":{"title":"...","input":{"text":"..."},"outputContract":{"text":"..."},"acceptance":{"rules":["..."]}}}
- Parser 只有在 itemId 精确匹配、workUnit 字段非空、acceptance.rules 是非空 string[]、且没有禁用字段时才返回 ok:true。
- Parser 对 invalid JSON、item mismatch、invalid schema、forbidden fields 必须返回 ok:false，不要 throw。
- Forbidden fields 至少包括 workerAgentId、checkerAgentId、leaderAgentId、generatedWorkerAgentId、generatedCheckerAgentId、canvasKind、discoverySpec、generatedSource、sourceDiscoveryTaskId、sourceItemId、itemPayload、itemStatus、workUnitMode、outputPorts、outputCheck。
- Prompt 必须包含 Discovery task id/title、Discovery goal、dispatch goal、outputKey、required/recommended item fields、exact item id、完整 item payload JSON、严格 JSON schema、以及不得输出 worker/checker/source identity 字段的硬约束。
